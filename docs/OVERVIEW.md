# Visão Geral

Minha Loja é uma plataforma de e-commerce desenvolvida com arquitetura profissional, com foco em escalabilidade, organização de código e boas práticas utilizadas em empresas de tecnologia.

O sistema simula um ambiente real de loja virtual, permitindo autenticação segura, gerenciamento de produtos e integração entre frontend e backend via API REST.

---

## Objetivos do projeto

Demonstrar:

- arquitetura escalável
- organização em camadas
- autenticação segura
- integração frontend e backend
- padrão profissional de desenvolvimento

---

## Escopo

O sistema contempla:

- autenticação de usuários
- gerenciamento de produtos
- carrinho de compras
- favoritos
- painel administrativo
- API REST versionada
