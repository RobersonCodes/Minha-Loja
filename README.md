# 🛒 Minha Loja  
### Plataforma de E-commerce Profissional (estilo Amazon)

Aplicação Full Stack desenvolvida com arquitetura profissional, backend escalável e experiência moderna inspirada em grandes plataformas de e-commerce.

Projeto criado para demonstrar boas práticas de desenvolvimento utilizadas no mercado.

---

## 🚀 Visão Geral

Minha Loja é um sistema completo de e-commerce desenvolvido para portfólio profissional.

O projeto simula uma loja online moderna com:

- autenticação segura
- arquitetura escalável
- API REST profissional
- frontend moderno
- organização de código corporativa

---

## ✨ Funcionalidades

### 🔐 Autenticação
- login com JWT
- senha criptografada com bcrypt
- rotas protegidas

### 📦 Produtos
- cadastro de produtos
- edição de produtos
- exclusão de produtos
- listagem de produtos
- categorias
- produtos em destaque

### 🛍 Loja
- carrinho de compras
- favoritos
- layout moderno
- armazenamento local

### 🛠 Admin
- painel administrativo
- métricas do sistema
- gerenciamento de produtos

## 📡 API

Base URL

/api/v1

Auth
POST /auth/register  
POST /auth/login  

Produtos
GET /products  
GET /products/:id  
POST /products  
PUT /products/:id  
DELETE /products/:id  

Admin
GET /admin/metrics

## ▶️ Como executar

instalar dependências

npm install

executar servidor

npm run dev

abrir frontend

frontend/pages/index.html
